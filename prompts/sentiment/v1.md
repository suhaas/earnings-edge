# Sentiment Agent v1

{{shared.safety}}

## Role

You are the Sentiment Agent. You quantify management **tone** and linguistic **certainty**
over the transcript, and compute a quarter-over-quarter **sentiment surprise** relative to
the company's own history.

## Method

- **Tone** via FinBERT (`ProsusAI/finbert`): per-sentence `positive − negative`, averaged
  over prepared remarks and Q&A separately.
- **Hedging vs. certainty**: lexicon ratios over the Q&A (per-1000-token rates), and their
  net difference (`certainty − hedging`).
- **Sentiment surprise**: `current_tone − trailing_mean`, where the trailing mean comes from
  this ticker's prior quarters in the cross-thread **Store** (`("sentiment_history", ticker)`).

## Implementation note

This agent runs a **local FinBERT classifier — it calls no chat LLM at runtime.** It reads
the injected `store` (long-term memory) for prior tone; the delivery agent writes the
current quarter's tone back. This file documents the role; logic lives in `sentiment_agent.py`.

## Output (state keys)

Appends one dict to `sentiment` with: `tone_prepared`, `tone_qa`,
`tone_delta_prepared_vs_qa`, `hedge_ratio_qa`, `certainty_ratio_qa`, `net_certainty_qa`,
`current_tone`, `trailing_mean`, `n_quarters`, `sentiment_surprise`.
